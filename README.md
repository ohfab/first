# Flappy Bird Demo

This repository contains a simple Flappy Bird clone implemented with `pygame`.

## Requirements

- Python 3
- pygame (`pip install pygame`)

## Running the Game

```bash
python flappy.py
```

Press the space bar to flap. Avoid the pipes and try to get the highest score!

## Context Vocabulary Prototype

This repo now also contains `context_vocab_app.py`, a small prototype for a location-based language learning app. It serves vocabulary lists for simple contexts.

### Running the Prototype

```bash
python context_vocab_app.py --context cafe
```

Replace `cafe` with another context like `supermarket` or `train_station` to see different vocab lists.
