# Context-Based Language Learning App

This repository lays out the initial directory structure for a location-aware language learning application.

## Directory Structure

- `backend/` – Placeholder for server-side code and APIs.
- `frontend/` – Placeholder for web-based user interfaces.
- `mobile/` – Placeholder for mobile application code.
- `docs/` – Documentation and project planning materials.
- `tests/` – Test suites for the project.

Each directory currently contains a `.gitkeep` file so the empty folders are tracked in version control.
