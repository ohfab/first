# Flappy Bird Demo

This repository contains a simple Flappy Bird clone implemented with `pygame`.

## Requirements

- Python 3
- pygame (`pip install pygame`)

## Running the Game

```bash
python flappy.py
```

Press the space bar to flap. Avoid the pipes and try to get the highest score!

## Personal Planner

A small command-line tool for taking notes and planning tasks for school.

### Usage

```bash
# Add a note or task
python planner.py add "Read chapter 1" --due 2024-09-01

# List all notes and tasks
python planner.py list

# Mark item as done
python planner.py done 1
```
